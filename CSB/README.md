# 🚀 Web3 Wallet dApp


This is a decentralized wallet built using React and Ethers.js.
It allows users to interact with a smart contract to deposit, withdraw, and transfer ETH.


- Connect MetaMask wallet
- Deposit ETH
- Withdraw ETH
- Transfer ETH to another address
- View balance
- Modern UI with animations


- React (Vite)
- Ethers.js
- Solidity
- MetaMask


npm install
npm run dev
